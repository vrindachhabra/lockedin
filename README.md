# LockedIn

**Your life. Organized.**

LockedIn is a full-stack productivity platform that helps users manage goals, tasks, placement preparation, and AI-generated planning systems.

## Features

### Productivity System

* Secure JWT authentication
* Daily Goals workspace with tasks, reminders, priorities, streaks, and recurring schedules
* List, Kanban, and Calendar views
* Placement Tracker for applications, OAs, interviews, deadlines, and status tracking
* Search, analytics, notifications, autosave drafts, and offline support

### AI Workspace Generator

* Create custom workspaces using natural-language prompts
* Generate systems for DSA prep, study plans, finance tracking, habit building, startup management, and more
* Dynamic dashboards with widgets, analytics, checklists, progress tracking
* AI-powered workspace refinement and MongoDB persistence

## Tech Stack

* React, TypeScript, Vite
* Tailwind CSS, Framer Motion
* Zustand, React Hook Form, Zod
* Node.js, Express
* MongoDB, Mongoose
* JWT Authentication
